How to run parser.py
1. Download tika-python from https://github.com/chrismattmann/tika-python/archive/master.zip
2. Inside tika-python-master directory, run commands:
	python setup.py build
	python setup.py install
3. Command to run script: python parser.py [insert file name here]
	ie: python parser.py badwriting.pdf
4. An output txt file should appear in same directory as location of input file
